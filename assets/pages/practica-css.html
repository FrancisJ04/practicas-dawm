Hipervínculos: abrir en una nueva pestaña

Atributo target="_blank": Este atributo se utiliza para especificar cómo se debe abrir el hipervínculo. Puede ser _blank, _self, _parent, o cualquier nombre diferente que no sea de los cuatro mencionados, que se considera _blank para abrir en una nueva pestaña o ventana.

Hipervínculos: abrir en la misma pestaña

Los hipervínculos permiten a los usuarios navegar entre diferentes páginas web de manera sencilla. Sin embargo, no todos los navegadores permiten abrir enlaces en la misma pestaña. Para aquellos que desean mantener la navegación dentro de su sitio web o evitar que los usuarios se distraigan con otras páginas externas, es posible utilizar HTML y la propiedad target del elemento de ancla para abrir un enlace en la misma pestaña. Esto se logra estableciendo el valor «_self» para el atributo «target» del enlace.

Abrir en la misma pestaña
Hipervínculos: abrir en la misma pestaña, modo seguro

Atributo rel="noopener noreferrer": Este es el uso combinado de los atributos noopener y noreferrer, que se utilizan para mejorar la seguridad y la privacidad al abrir hipervínculos. Al usar estos atributos, se puede garantizar que los hipervínculos se abren en una nueva pestaña o ventana, proporcionando una experiencia de navegación más segura y responsable
